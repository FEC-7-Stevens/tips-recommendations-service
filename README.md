## Usage

>Example URL: http://localhost:6070/1/

To initialize the page, run the following commands in two separate terminal windows:
```sh
npm run server
npm run build
```

## Requirements

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Database Creation

From within the root directory:

```sh
npm run schema
npm run seed
```
